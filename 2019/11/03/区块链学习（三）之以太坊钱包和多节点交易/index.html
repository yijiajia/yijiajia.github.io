一、以太坊钱包

我们可以通过命令行进行转账，但这样做不好操作且不直观。可通过以太坊的钱包进行转账。

以太坊钱包的种类

1. Ethereum官方钱包+浏览器 Mist

Ethereum-Wallet是稳定版的钱包客户端，Mist是一个全节点钱包，全节点钱包通俗的来说就是同步了全部的以太坊区块信息的钱包，也就是说打开钱包后，电脑会自动同步全部的以太坊区块信息。

2. Parity

原以太坊基金会部分成员，开发的钱包。功能强大，也是一个全节点钱包。

3.MyEtherWallet

MyEtherWallet作为一个轻钱包，是一个网站，无需下载，在直接在网页上就可以完成所有的操作。不能链接私链

4. imToken

这是大部分都很熟悉的钱包了，由杭州的团队开发，移动包括android，iOS，操作简便，容易上手，功能齐全，在imToken上生成的钱包私钥保存在手机本地，平台方并无备份。

5. MetaMask

Chrome浏览器的插件，大名鼎鼎，可以链接以太坊私链

Mist浏览器的下载与配置

  • 下载地址：https://github.com/ethereum/mist/releases/

  • 安装Mist：解压Mist到指定文件夹,并添加到Path环境变量

配置后可在命令行输入Mist启动钱包，也可以直接安装目录下的Mist.exe

打开Mist后会出现两种情况，当已经在终端启动了节点，那么Mist会自动连接到节点上；若没启动，Mist会调用自己下载的geth启动节点（下载的geth在该目录下：C:\Users\keke\AppData\Roaming\Mist\binaries\Geth）。

Mist钱包界面的功能：

  • 钱包账户管理：显示余额，新建账户
  • 转账功能
  • 智能合约功能
  • 最近交易记录的显示
